# 方块塔防 3D · FPS 版 🎯

一个用 **Three.js** 写的浏览器端 3D 第一人称射击塔防游戏，零外部资源，单文件即开即玩。

## 🎮 操作

| 输入 | 行为 |
|------|------|
| `W` `A` `S` `D` / 方向键 | 第一人称移动 |
| `空格` | 跳跃 |
| `Shift` | 加速冲刺 |
| 鼠标移动 | 瞄准 |
| 鼠标左键 | 射击（**按住可连射**） |
| `1` `2` `3` `4` / 滚轮 | 切换武器 |
| `B` | 打开 / 关闭商店（自动暂停） |
| `Esc` | 退出指针锁定 / 关闭商店 |

> ⚠️ 注意：原版本"空格 = 开火"，现在 **空格 = 跳跃**，开火请用鼠标左键。

## ✨ 本版本亮点

### 🏃 第一人称玩家系统
- WASD 移动、Shift 冲刺、空格跳跃，**自带重力**
- 受击时屏幕红屏闪烁 + 击退
- 不能穿过基地、不能掉出地图
- 敌人不再傻冲基地，而是 **追着玩家打**

### 🔫 4 种武器
| 武器 | 快捷键 | 特点 |
|------|--------|------|
| 手枪 | `1` | 默认武器，平衡 |
| 散弹枪 | `2` | 一次 5 发扇形散射，近距离爆杀 |
| 激光 | `3` | 即时命中（射线检测），射速极高 |
| 火箭弹 | `4` | 范围爆炸（AOE），收割成群敌人 |

切换：数字键 `1-4`、鼠标滚轮，或点击底部武器栏。

### 💰 商店与升级
按 `B` 打开商店（自动暂停），可购买：
- 💥 伤害提升 +25%（可叠加）
- ⚡ 射速提升 +20%（可叠加）
- ❤️ 修复基地 +30 血
- 🛡️ 基地血量上限 +25
- 🔓 解锁散弹枪 / 激光 / 火箭弹

金币来源：**击杀敌人 + 通关每波奖励**。

### 🐲 敌人种类
| 类型 | 特征 | 出现 |
|------|------|------|
| 普通方块 | 标准速度，1 血 | 第 1 波起 |
| 快速小兵 | 体型小、速度极快 | 第 2 波起 |
| 飞行单位 | 浮空，需仰头瞄准 | 第 3 波起 |
| 坦克 | 大体型、高血量、撞击伤害高 | 第 4 波起 |
| **BOSS** | 巨大、50 血、爆炸性入场 | **每 5 波一次** |

### 🎵 音效 + BGM
- 全部由 Web Audio API **程序化合成**，不依赖任何外部音频文件，体积零增加
- 包含：开火、散弹、激光、火箭、爆炸、命中、击杀、金币、受伤、波次、购买等
- BGM 是 8 拍循环的 A 小调 chiptune（贝斯 + 琶音）

### 🛠 兼容性
- **Pointer Lock 失败时自动降级**：如果当前环境（如某些 IDE 内置 WebView）不允许指针锁定，会自动切换为"鼠标在画面上移动即瞄准"，照样能玩
- 浏览器 / Chromium 内置预览 / VSCode Live Server 全部支持

## 🚀 运行方式

### 方法 1：双击打开
直接双击 `index.html`。

### 方法 2：本地服务器（更稳）
```powershell
npx --yes serve . -l 8765
# 或
python -m http.server 8080
```

### 方法 3：VSCode Live Server 插件
右键 `index.html` → "Open with Live Server"。

## 💡 进阶玩法建议

- 优先把"伤害"和"射速"各升 1-2 级再考虑解锁高级武器
- 火箭弹千万别冲着自己脚下打（虽然没设自伤，但浪费）
- 看到飞行单位记得仰头
- BOSS 出场时立刻切散弹或火箭，集中输出
- 每波结束有 30 金奖励，省着点花
